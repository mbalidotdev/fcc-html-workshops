# HTML Video Player

A simple html video player that uses different video MIME (Multipurpose Internet Mail Extensions) types, which is a standard to describe documents in other forms.

## Features

- Video player with different attributes:
  - `controls`
  - `muted`
  - `loop`
  - `width`
  - `poster`
- MIME types
  - `video/mp4`
  - `video/webm`
  - `video/ogg`
  - `video/quicktime`

## Technologies

- HTML5
